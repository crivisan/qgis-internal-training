# Georeferencing & Layout

## Aufgabe

Erstellen Sie eine vollständige Karte mit einer georeferenzierten historischen Karte.

### Teil 1: Georeferencing

1. Laden Sie eine gescannte alte Karte (stellen Sie selbst bereit oder nutzen Sie eine aus dem Kurs)
2. Öffnen Sie den Georeferencer
3. Setzen Sie mindestens **6 GCPs** gleichmäßig verteilt
4. Führen Sie die Transformation durch (Helmert oder Polynomisch 1)
5. Exportieren Sie als `georef_karte.tif`


### Teil 2: Print Layout

1. Erstellen Sie ein neues Drucklayout
2. Fügen Sie die georeferenzierte Karte ein
3. Stellen Sie einen passenden Maßstab ein
4. Fügen Sie einen Nordpfeil ein
5. Fügen Sie eine Maßstabsleiste ein
6. Fügen Sie eine Legende ein
7. Fügen Sie einen Titel ein (z.B. `Meine erste georeferenzierte Karte`)
8. Exportieren Sie das Layout als **PDF**

## Reflexionsfragen

<details>
<summary><strong>Frage 1: Warum braucht man mindestens 4 GCPs?</strong></summary>

Eine Helmert-Transformation benötigt mindestens 2 Punkte für Verschiebung, Skalierung und Rotation. Mehr Punkte (4-6) verbessern die Genauigkeit und erlauben eine Fehlerkontrolle (RMS).

</details>

<details>
<summary><strong>Frage 2: Welche Layout-Elemente gehören zu einer vollständigen Karte?</strong></summary>

- Karte (Hauptelement)
- Maßstabsleiste
- Nordpfeil
- Legende
- Titel
- ggf. Maßstabszahl, Datum, Quellenangabe

</details>
