# Sitzung 3 – Georeferencing & Drucklayout

## Lernziele dieser Sitzung

Nach dieser Sitzung können Sie:

- Eine gescannte Karte oder ein Luftbild georeferenzieren
- Ground Control Points (GCPs) setzen
- Das Georeferencer-Werkzeug in QGIS bedienen
- Ein Drucklayout erstellen
- Karten mit Nordpfeil, Maßstabsleiste und Legende versehen
- Karten als PDF oder Bild exportieren

## Aufbau der Sitzung

| Block | Thema |
|-------|-------|
| 1 | Georeferencing – gescannte Karten ausrichten |
| 2 | Print Layout – Kartenlayout erstellen und exportieren |
| 3 | Übung – eigene Karte georeferenzieren und layouten |

## Was Sie für diese Sitzung brauchen

- QGIS installiert
- Eine gescannte Karte oder ein Luftbild (z.B. historische Karte, Stadtplan)
- Optional: OpenStreetMap als Referenz im Hintergrund
- Ca. 1,5 Stunden Zeit
