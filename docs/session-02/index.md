# Sitzung 2 – Daten laden und externe Dienste

## Lernziele dieser Sitzung

Nach dieser Sitzung können Sie:

- Lokale Geodaten in QGIS laden (Shapefile, CSV, GeoPackage)
- CSV-Dateien als Punktlayer importieren
- Attributtabellen öffnen und einfache Filter anwenden
- WMS- und WFS-Dienste unterscheiden
- Webdienste in QGIS einbinden
- Das Plugin `geobasis_loader` kennenlernen

## Aufbau der Sitzung

| Block | Thema |
|-------|-------|
| 1 | Lokale Daten laden (Shapefile, CSV, Export in andere Formate) |
| 2 | Attributtabellen und einfache Abfragen (kurze Demo) |
| 3 | Externe Dienste (WMS/WFS) und Plugin-Vorstellung |
